### 헷갈리는 마크다운 문법 몇 가지

#### 1. 코드 블록 (codeblock)

코드 블록을 표시하기 위해 쓰이는 **`** 는 ~(물결표) 아래에 있는 **grave** 라고 함.  
(작은 따옴표와 혼동하지 말 것.)

![grave, 작은 따옴표](https://github.com/upqnu/TIL/blob/main/_image/grave.jpg)
  
#### 2. 줄 바꿈 (개행)

마크다운 작성할 때 enter를 쳐서 줄을 바꿔도 실제로 줄이 바뀌지 않음.  
이럴 때는 문장의 마지막에 `space` 키를 2번 치면 된다.
